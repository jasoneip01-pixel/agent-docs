⚡ Quick Start

Clone & Install

$ git clone https://github.com/jasoneip01-pixel/agent-datadog
$ cd agent-datadog
$ pip install -r requirements.txt
$ uvicorn main:app --reload

Cost Compare API

$ curl https://api.agent-datadog.dev/api/cost/compare?model=gpt-4o

Benchmark API

$ curl https://api.agent-datadog.dev/api/cost/benchmark

Open Dashboard

$ open http://localhost:8000
# SSE live stream · cost trends · latency heatmap · comparison matrix

Find Best Value Model

$ curl -s https://api.agent-datadog.dev/api/cost/compare?model=gpt-4o \
  | jq '.best_value'

Compare Across Frameworks

$ curl -s https://api.agent-datadog.dev/api/cost/benchmark \
  | jq '.recommendations'

🛡️ What It Does

💰

Cross-Framework Cost Comparison

Compare input/output token pricing across OpenAI, Anthropic, DeepSeek, Google, and Mistral. Query by model name, get side-by-side cost, latency, and error-rate data.

📊

Benchmark Engine

Full benchmark matrix for all 14 models. Sortable by cost, latency, or composite score. Weighted scoring: cost 40%, latency 30%, reliability 30%.

📡

SSE Live Stream

Real-time event stream via Server-Sent Events. Cost updates, latency alerts, and error spike notifications pushed directly to your dashboard.

📈

Dashboard Panel

Vanilla JS dashboard with cost trends, latency heatmap, error rate tracking, and framework comparison matrix. No build tools required.

🏆

Best Value Detection

Context-aware best-value recommendation. Not just the cheapest model — the best cost/latency/reliability tradeoff for your use case.

🔌

Datadog Exporter

Push metrics to Datadog APM for unified observability. Histogram, gauge, and event types supported. Integrate with your existing monitoring stack.

❓ FAQ

How is this different from AgentWatch?
AgentWatch is a single-framework monitoring tool for OpenAI calls. Agent Datadog forks AgentWatch and adds a cross-framework comparison layer — cost/latency/error observability across 5 frameworks and 14 models. AgentWatch answers "Is my agent healthy?" Agent Datadog answers "Which agent setup costs me the least?"
Why not just use LangSmith or Helicone?
LangSmith is tightly coupled to LangChain. Helicone proxies OpenAI calls. Neither provides cross-model cost comparison. When you're deciding between gpt-4o, claude-3.5-sonnet, and deepseek-chat — which saves you the most? Agent Datadog's benchmark engine answers this directly.
Is it open source?
Yes. MIT license. Fork of MIT-licensed AgentWatch (364 lines of FastAPI). The cross-framework comparison layer is our addition.
What's the roadmap?
Phase 1 (✅ done): Core dashboard with SSE stream, cost/latency/error monitoring. Phase 2 (🚧): Dashboard comparison panel with visual model selector and side-by-side charting. Phase 3 (🔲): agent-cost CLI integration and Datadog exporter.
Can I add my own framework?
Yes. The pricing table is a simple data structure. Add a new framework with model names, pricing, and benchmark data — the comparison engine works automatically.
How is the composite score calculated?
Weighted scoring: cost per 100k tokens (40%) + average latency (30%) + reliability = 1 - error_rate (30%). All dimensions normalized to 0-10 scale. The weights are configurable.
